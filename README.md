# 바톤 (BATON) — 업무 인수인계 AI 챗봇

> "바통을 넘기듯, 업무도 끊김 없이"

---

## 프로젝트 소개

**바톤**은 슬랙 기반의 AI 업무 인수인계 챗봇입니다.
신입직원과 기존 재직자 모두가 업무를 빠르게 파악하고 체득할 수 있도록 돕습니다.

- "김신성이라면 이렇게 대답했을 거예요" 방식으로 전임자의 판단 기준을 학습
- 슬랙에서 바로 질문하고 답변받는 자연스러운 업무 연계
- 마지막 단계에서 실전 시뮬레이션으로 실제 대응 능력 훈련

---

## 대상 사용자

| 대상 | 상황 |
|------|------|
| 신입직원 | 처음 업무를 배우는 단계 |
| 기존 재직자 | 담당 업무 변경 또는 인수인계 받는 상황 |

---

## 플랫폼

- **슬랙(Slack)** — 기존 업무 환경과 연계

---

## 작동 방식 (3단계)

### 1단계: 세팅
전임자의 업무 방식, 자주 묻는 질문, 판단 기준 등을 바톤에 입력

### 2단계: 온보딩 학습
슬랙에서 질문하면 바톤이 전임자 스타일로 답변
> "김신성이라면 이렇게 대답했을 거예요 :)"

### 3단계: 실전 시뮬레이션
AI 봇이 실제 훈련생 페르소나(20명, 강성 포함)로 DM·스레드에 등장
담당자가 직접 대응하며 실전 감각 체득
