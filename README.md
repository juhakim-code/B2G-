## 폴더 구조

```
docs/
├── 문제정의/     ← 우리가 풀려는 문제
│   ├── 김주하.md
│   ├── 신성.md
│   └── 재은.md
├── 솔루션/       ← 우리의 해결책
│   ├── 김주하.md
│   ├── 신성.md
│   └── 재은.md
└── GTM/          ← 시장 진입 전략
    ├── 김주하.md
    ├── 신성.md
    └── 재은.md
```

## 작성 방법

1. 내 이름의 파일을 열어 작성한다
2. 저장 후 커밋 & 푸시한다
3. 다른 팀원 파일도 읽고 피드백을 남긴다
