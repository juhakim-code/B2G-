# 바톤 (BATON) — 업무 인수인계 AI 챗봇

> "바통을 넘기듯, 업무도 끊김 없이"

---

## 프로젝트 소개

**바톤**은 슬랙 기반의 AI 업무 인수인계 챗봇입니다.
신입직원과 기존 재직자 모두가 업무를 빠르게 파악하고 체득할 수 있도록 돕습니다.

- "김신성이라면 이렇게 대답했을 거예요" 방식으로 전임자의 판단 기준을 학습
- 슬랙에서 바로 질문하고 답변받는 자연스러운 업무 연계
- 마지막 단계에서 실전 시뮬레이션으로 실제 대응 능력 훈련

---

## 대상 사용자

| 대상 | 상황 |
|------|------|
| 신입직원 | 처음 업무를 배우는 단계 |
| 기존 재직자 | 담당 업무 변경 또는 인수인계 받는 상황 |

## 작동 방식 (3단계)

### 1단계: 학습 (`/바톤-학습`)

슬랙 전체 채널 메시지를 읽어 Claude API로 분석합니다.

```
슬랙 채널 메시지
    → Claude가 아래 3가지로 자동 분류
        ├── 업무 프로세스
        ├── 자주 묻는 질문
        └── 담당자 스타일 (말투·판단 기준)
    → 지식창고(knowledge_store)에 저장
```

> 매주 월요일 오전 6시 자동 실행 (APScheduler)

---

### 2단계: 인계 (`/바톤-인계` → `/바톤-인계-확정`)

전임자가 인계할 항목을 선택하면 인수자에게 자동 전달합니다.

```
/바톤-인계 실행
    → Claude가 지식창고에서 인계 후보 10개 생성 (우선순위: 상/중/하)
    → 전임자 DM으로 목록 전송

/바톤-인계-확정 @인수자 1,2,3 실행
    → 선택한 항목으로 온보딩 문서 생성
    → 인수자 DM으로 자동 발송
```

---

### 3단계: 실전 시뮬레이션 (`/baton-simulation`)

AI 페르소나 20명이 채널에 등장해 실전처럼 질문합니다.

```
/baton-simulation 20 실행
    → 페르소나 20명이 순서대로 채널에 질문 게시
    → 담당자가 스레드에 답변
        → 페르소나가 성격에 맞게 반응
        → 바톤이 AI 피드백 자동 제공 (점수·잘한점·보완점·모범답변)
```

**페르소나 구성 (20명)**

| 유형 | 인원 | 예시 |
|------|------|------|
| 🔴 고난도 | 5명 | 김급함, 최불만, 정복잡, 한예외, 오반복 |
| 🟢 일반 | 15명 | 박신입, 이조용, 강열심, 윤전직 등 |

---

### DM 질문 기능

바톤 봇에게 DM으로 언제든 질문할 수 있습니다.

```
사용자 DM: "출결 정정은 어떻게 처리해요?"
바톤 답변: "김신성이라면: [전임자 스타일 답변]"
```
