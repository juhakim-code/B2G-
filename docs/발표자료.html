04   실무 적용 시나리오

누가, 언제, 어떻게 씁니까

CASE 1  ·  신입 온보딩

첫 출근 당일부터

장재은(신입)이 바톤봇 DM에 질문
→ 김신성 스타일로 즉시 답변
→ 선배 눈치 없이 자립 온보딩
✓ 온보딩 기간 3개월 → 1개월 목표
CASE 2  ·  담당자 교체

인수인계 당일 완료

전임자가 /바톤-인계 실행
→ 핵심 업무·판단기준 자동 추출
→ 후임자에게 맞춤 인계서 즉시 전달
✓ 퇴직 후 카카오톡 문의 Zero
CASE 3  ·  실전 시뮬레이션

혼자서 민원 대응 연습

AI 페르소나 20명이 DM 발송
→ 강성 민원·반복 질문 포함
→ 실전과 동일한 대응 감각 체득
✓ 실전 투입 전 리허설 가능
CASE 4  ·  지식 자산화

매주 자동으로 업데이트

매주 월요일 자동 학습 스케줄러
→ 최신 채널 대화 반영
→ 조직의 살아있는 지식 DB
✓ 담당자 바뀌어도 지식은 남는다
